鸡尾酒配方

| 调酒                           | 配料                                               | 比例                                                         | 方法    |
| ------------------------------ | -------------------------------------------------- | ------------------------------------------------------------ | ------- |
| **干马天尼 Dry Martini**       | 金酒, 干味美思                                     | 6份金酒, 1份干味美思<br />仪式：马天尼杯                     | Stir    |
| **斯普莫尼 Spumoni**           | 金巴利, 西柚汁, 汤力水                             | 1份金巴利, 2份西柚汁, 3份汤力水                              | Shake   |
| **威士忌酸 Whiskey Sour**      | 威士忌, 柠檬汁, 糖                                 | 3份威士忌, 2份柠檬汁, 1份糖                                  | Shake   |
| **螺丝锥 Gimlet**              | 金酒, 青柠汁                                       | 4份金酒, 1份青柠汁                                           | Shake   |
| **新加坡司令 Singapore Sling** | 金酒, 樱桃利口酒, 君度, 菠萝汁, 青柠汁, 石榴糖浆   | 3份金酒, 1份樱桃利口酒, 1/2份君度, 1/2份菠萝汁, 1/4份青柠汁, 1/4份石榴糖浆 | Shake   |
| **薄荷朱莉普 Mint Julep**      | 波本威士忌, 薄荷, 糖, 水                           | 2份波本威士忌, 8片薄荷叶, 1茶匙糖, 2茶匙水                   | Stir    |
| **迈泰 Mai-Tai**               | 朗姆酒, 青柠汁, 橙味白兰地, 杏仁糖浆               | 2份朗姆酒, 1份青柠汁, 1/2份橙味白兰地, 1/2份杏仁糖浆         | Shake   |
| **盘尼西林 Penicillin**        | 苏格兰威士忌, 柠檬汁, 蜂蜜, 姜                     | 2份苏格兰威士忌, 3/4份柠檬汁, 3/4份蜂蜜, 1/4份姜             | Shake   |
| **玛格丽特 Margarita**         | 龙舌兰, 青柠汁, 君度                               | 3份龙舌兰, 2份青柠汁, 1份君度<br />仪式：杯口一圈盐          | Shake   |
| **血腥玛丽 Bloody Mary**       | 伏特加, 番茄汁, 柠檬汁, 伍斯特酱, 辣椒酱, 盐, 胡椒 | 1份伏特加, 3份番茄汁, 1/2份柠檬汁, 1滴伍斯特酱, 辣椒酱适量, 一撮盐, 一撮胡椒<br />仪式：关灯端着酒对着浴室的镜子说三声“I believe bloodymary”即可召唤血腥玛丽 | Rolling |
| **莫斯科骡子 Moscow Mule**     | 伏特加, 生姜啤, 青柠汁, 装饰用青柠片和薄荷叶       | 2份伏特加, 6份生姜啤, 1/2份青柠汁, 适量装饰用青柠片和薄荷叶  | Stir    |
| **教父 Godfather**             | 苏格兰威士忌, 杏仁利口酒                           | 1份苏格兰威士忌, 1份杏仁利口酒                               | Stir    |
| **自由古巴 Libre**             | 朗姆酒, 可乐, 青柠汁                               | 2份朗姆酒, 4份可乐, 1/2份青柠汁                              | Stir    |
| **黑俄罗斯 Black Russian**     | 伏特加, 咖啡利口酒                                 | 2份伏特加, 1份咖啡利口酒                                     | Stir    |
| **白俄罗斯 White Russian**     | 伏特加, 咖啡利口酒, 奶油                           | 2份伏特加, 1份咖啡利口酒, 1份奶油                            | Stir    |
| **边车 Sidecar**               | 干邑白兰地, 君度, 柠檬汁                           | 2份干邑白兰地，1份君度，1份柠檬汁                            | Shake   |
| **大都会Cosmopolitan**         | 伏特加，君度，柠檬汁，蔓越莓汁                     | 3份伏特加, 1份君度, 2份柠檬汁, 2份蔓越莓汁                   | Shake   |
